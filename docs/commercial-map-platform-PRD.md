
# 지도 기반 상가 매물 관리 플랫폼 — 상세 PRD

**Version:** 1.0  
**Owner:** Client  
**작성 목적:** 대구 지역 상가 중개업무 효율화를 위한 지도 기반 매물 관리 플랫폼 구축

---

## 1. Product Vision
대구 지역 상가 중개 시장에서  
**“지도 중심의 직관적인 탐색 + 권한 기반 민감정보 통제 + 파트너 참여형 매물 관리 시스템”**을 제공하여  
업무 효율을 극대화하고, 신뢰 기반의 전문 중개 환경을 구축한다.

---

## 2. Target Users

### 2.1 사용자 유형 정의

| 유형 | 설명 | 권한 수준 |
|------|------|--------|
| 비회원 | 단순 탐색 사용자 | 제한적 정보 조회 |
| 일반 회원 | 가입 완료 사용자 | 기본 정보 조회 |
| 인증 회원 (직원) | 내부 운영 인력 | 전체 정보 조회 + 관리 |
| 파트너 회원 | 승인된 제휴 중개사 | 매물 등록/관리 가능 |
| 마스터 관리자 | 시스템 운영자 | 전체 권한 |

---

## 3. 주요 목표

1️⃣ 지도 기반 상가 매물 탐색 제공  
2️⃣ 민감 정보(권리금 등) 등급별 노출 제어  
3️⃣ 내부 + 파트너 매물 관리 통합 플랫폼 구축  
4️⃣ 결제 없음 / 권한 수동 승인 구조 유지  
5️⃣ 반응형 웹 기반 구축, 모바일 앱은 추후 옵션

---

## 4. 서비스 기능 정의

---

### 4.1 지도 기반 매물 탐색 시스템

#### 기능 목적
사용자가 직관적으로 위치 기반으로 상가 매물을 탐색할 수 있도록 지원

#### 상세 기능
- 지도 API 연동
- 지도 확대/축소 / 이동
- 클러스터링 적용
- 매물 핀 표시
- 매물 요약 툴팁 표시
- 리스트형 / 지도형 제공

#### 검색 / 필터 조건
- 지역 (구 / 동 / 반경)
- 매물 유형
- 보증금 / 월세
- 권리금 (권한 제한)
- 면적
- 상태
- 등록자 타입

---

### 4.2 등급별 정보 열람 제한

| 정보 항목 | 비회원 | 일반회원 | 인증/파트너 | 관리자 |
|---------|-------|--------|------------|--------|
| 기본 정보 | O | O | O | O |
| 상세 사진 | O | O | O | O |
| 상권 데이터 | X | △ | O | O |
| 권리금 | X | X | O | O |
| 중개자 정보 | X | O | O | O |

---

### 4.3 매물 등록 및 관리 시스템

#### 등록 가능 대상
- 관리자
- 인증 직원
- 승인 파트너

#### 등록 항목
- 기본 정보
- 주소 (좌표 자동 변환)
- 임대 조건
- 면적
- 층수
- 업종 가능 여부
- 상세 설명
- 사진 업로드
- 노출 설정

#### 관리 기능
- 수정 / 삭제
- 상태 변경
- 등록 이력 관리

---

### 4.4 사용자 권한 및 멤버십 관리

- 회원 가입
- 파트너 승인 프로세스
- 등급 변경 (관리자 수동 승인)
- 권한 정책 엔진 구축
- 접근 로그 기록

---

### 4.5 반응형 웹
- Desktop: 좌측 리스트 + 우측 지도
- Mobile: 지도 우선 + 리스트 슬라이드
