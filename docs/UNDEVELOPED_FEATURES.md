# 미구현 기능 목록

## 🔴 필수 (PRD 요구사항)

### 1. 반경 검색 기능
- **상태**: ❌ 미구현
- **우선순위**: 높음
- **설명**: 중심점 좌표와 반경(km)을 입력하여 해당 영역 내 매물 검색
- **예상 공수**: 4시간

### 2. 감사 로그 조회 UI
- **상태**: ⚠️ 부분 구현 (트리거는 존재, UI 없음)
- **우선순위**: 높음
- **설명**: 관리자 페이지에서 감사 로그 조회 및 필터링
- **예상 공수**: 3시간

---

## 🟡 권장 (선택사항)

### 3. 상세 등록 이력 UI
- **상태**: ⚠️ 부분 구현 (감사 로그는 있지만 UI 없음)
- **우선순위**: 중간
- **설명**: 매물 상세 페이지에서 수정 이력 확인
- **예상 공수**: 2시간

---

## 🟢 선택 (추가 기능)

### 4. Redis 캐싱
- **상태**: ❌ 미구현
- **우선순위**: 낮음
- **설명**: 성능 최적화를 위한 캐싱 레이어
- **예상 공수**: 8시간

### 5. 장애 모니터링
- **상태**: ❌ 미구현
- **우선순위**: 낮음
- **설명**: 시스템 모니터링 및 알림
- **예상 공수**: 16시간

---

## ✅ 이미 구현된 항목

1. ✅ 매물 요약 툴팁 (hover 시 InfoWindow 표시)
2. ✅ 감사 로그 자동 기록 트리거 (audit_property_changes, audit_user_changes)
3. ✅ 클러스터링
4. ✅ GPS 위치 추적 (모바일 지속 추적)
